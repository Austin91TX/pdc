Amazon Poly

AboutX

The Poly Amazon Connect CCP implements Poly headset-based call control for Contact Center Agents using Amazon Connect.

Pre-requisites:

  1. Install Plantronics Hub: www.plantronics.com/software.
  2. Install Google Chrome: www.google.co.uk/chrome.
  3. Or Firefox: www.mozilla.org/en-GB/firefox/new/.
  4. If Amazon CCP (Contact Control Panel) “pops-out” to another Tab, sign in to that Tab then close it, then refresh main app window.
  5. Ready to use Poly headset buttons to answer/end/mute/un-mute calls!

Minimum resolution of 690x670 required.
Please expand your browser window.
